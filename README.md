# Knowledge Vault

This is an Obsidian vault tracked with Git and GitHub.

## Structure
- `inbox/` - temporary captures
- `sources/youtube/` - source notes from videos and podcasts
- `concepts/` - evergreen distilled knowledge
- `projects/` - active project notes
- `VAULT_INDEX.md` - human-readable vault map
- `vault_manifest.json` - machine-readable vault map

## Workflow
1. Capture or paste a source
2. Create/update source note
3. Update related concept/project notes
4. Refresh index files
5. Commit changes with Git
